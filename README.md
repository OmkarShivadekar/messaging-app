# messaging-app
Inbox app using Java, Spring Boot and Apache Cassandra.  This is full web application that is scalable and highly available. 

## Functional Requirements
-   compose message
-   send message
-   view my message
	- folder/label
	- inbox
	- sent
	- user folder
-   reply, reply all
-   view a single message

## Non functional requirements
- high availability
- high scalability
- authentication
- autharization
